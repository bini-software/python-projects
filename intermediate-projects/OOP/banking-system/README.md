## Simple Python Banking System

## Overview

This is a single-user banking system built in Python using Object-Oriented Programming (OOP) principles.
It simulates basic banking operations like deposit, withdrawal, balance check, and transaction history.

## Features

Create a bank account with account number, owner name, and starting balance

Deposit money into the account

Withdraw money from the account (with balance check)

Check current balance

View transaction history with timestamps


## Important Notes

This system supports only one user account per session.

No multi-user or networked functionality yet.

Input validation and error handling are implemented to ensure smooth user experience.


## How to Run

1. Clone or download this repository


2. Run main.py using Python 3.11+


3. Follow the prompts to create your account and perform banking operations



python main.py
